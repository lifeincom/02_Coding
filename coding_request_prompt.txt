코딩 요청용 프롬프트 템플릿
-------------------------------

당신은 전문 개발자입니다.
아래의 요구사항에 맞는 코드를 작성해주세요.

[프로젝트 목적]
(예: 엑셀 VBA로 작성된 재고 관리 매크로를 파이썬 GUI로 전환)

[기술 스택]
(예: Python, tkinter, openpyxl, tksheet)

[입력 항목]
- (예: 상품명, 단가, 수량)

[출력 결과]
- (예: 총금액, 마진율, 이익률)

[기능 요구사항]
1. (예: 입력값을 UI에서 받는다)
2. (예: 버튼 클릭 시 계산 수행)
3. (예: 결과를 표 형태로 표시)
4. (예: Excel 파일로 저장)

[UI 구성]
- 창 크기: 800x600
- 상단: 입력 필드
- 중앙: 결과 테이블
- 하단: 실행 및 저장 버튼
(UI 구성 요구사항이 없다면 생략 가능)

[추가 조건]
- 각 기능 블록마다 주석 추가
- 변수명은 직관적으로 작성
- 빈 입력값 있을 경우 오류 메시지 표시
- 실행 가능한 전체 코드 형태로 제공
- 초보자도 이해할 수 있도록 상세 주석 포함

[출력 형태]
- 하나의 완전한 코드 블록으로 제공
- 주석은 한국어로 작성
- 필요 시 예시 입력/출력 포함
