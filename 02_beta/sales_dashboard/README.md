# 판매 데이터 인사이트 대시보드 웹 애플리케이션

**버전**: 2.0  
**작성일**: 2026-02-09  
**작성자**: AI Assistant

---

## 📋 개요

기존 Python Tkinter 기반 판매 데이터 분석 도구를 **웹 기반 인사이트 대시보드**로 전환한 프로젝트입니다.

### 주요 기능

✅ **기존 기능 완벽 유지**

- 4가지 피벗 테이블 분석 (상품별, 쇼핑몰별, 일자별, 연도별)
- Excel 파일 로딩 및 Pickle 캐싱
- 날짜 범위 및 쇼핑몰 필터링
- Excel 내보내기

🆕 **신규 기능**

- **페이지네이션**: 페이지당 10/20/50/100/전체 행 선택 가능 (기본 20행)
- **판매 예측**: 향후 2주간 쇼핑몰별, 상품별 판매량 예측
- **대시보드 카드**: 주요 지표 시각화 (총 판매량, 상품 수, 상위 상품/쇼핑몰 등)
- **인터랙티브 차트**: Chart.js 기반 라인/바 차트
- **반응형 디자인**: 모바일 및 태블릿 대응
- **모던 UI/UX**: 깔끔하고 직관적인 디자인

## 📊 사용 방법

### 기본 분석

1. **날짜 범위 선택**
   - 시작일/종료일 직접 입력
   - 빠른 선택 버튼 사용 (1주, 2주, 1개월, 3개월, 6개월)

2. **쇼핑몰 필터**
   - 전체 선택 / 선택 해제 버튼
   - 개별 쇼핑몰 체크박스 선택

3. **페이지 설정**
   - 페이지당 표시할 행 수 선택 (10, 20, 50, 100, 전체)

4. **분석 실행**
   - "▶ 분석 실행" 버튼 클릭
   - 요약 카드, 차트, 테이블 자동 업데이트

### 탭별 기능

#### 📈 개요 탭

- 주요 지표 요약 카드 (총 판매량, 상품 수, 주문 건수 등)
- 일자별 판매 추이 라인 차트
- 쇼핑몰별 판매 비교 바 차트

#### 📦 상품별 분석

- 상품코드 × 일자 피벗 테이블
- 합계 기준 내림차순 정렬
- 페이지네이션 지원

#### 🏪 쇼핑몰별 분석

- 상품코드 × 쇼핑몰 피벗 테이블
- 합계 기준 내림차순 정렬

#### 📅 일자별 추이

- 일자 × 쇼핑몰 피벗 테이블
- 날짜 오름차순 정렬

#### 🗓️ 연도별 추이

- 년도 × 쇼핑몰 피벗 테이블
- 년도 내림차순 정렬 (최신 먼저)

#### 🔮 판매 예측

- "🔮 예측 실행" 버튼 클릭
- **쇼핑몰별 예측**: 향후 2주간 쇼핑몰별 판매량
- **상위 상품 예측**: 상위 10개 상품의 향후 2주 판매량
- 이동평균 + 선형회귀 + 요일패턴 복합 예측

### 기타 기능

- **💾 Excel 저장**: 현재 분석 결과를 Excel 파일로 다운로드
- **🔄 초기화**: 날짜 및 필터 설정 초기화

## 🧠 예측 알고리즘 설명

향후 14일간의 판매량을 다음 3가지 방법으로 예측한 후 가중 평균합니다:

### 1. 이동 평균 (40%)

- 7일 이동 평균
- 14일 이동 평균
- 가중 이동 평균 (최근 데이터에 더 높은 가중치)

### 2. 선형 회귀 (30%)

- 최근 30일 데이터로 추세선 계산
- 증가/감소 추세 반영

### 3. 요일별 계절성 (30%)

- 같은 요일의 과거 판매 패턴 분석
- 주말/평일 차이 반영

**최종 예측값 = 이동평균 × 0.4 + 선형회귀 × 0.3 + 요일패턴 × 0.3**

신뢰구간도 함께 제공됩니다 (표준편차 기반).

## 📝 API 엔드포인트

REST API 형태로 제공되어 외부에서도 사용 가능합니다.

### GET `/api/info`

데이터 정보 조회

### GET `/api/malls`

쇼핑몰 목록 조회

### POST `/api/analyze`

분석 실행

**요청 본문:**

```json
{
  "start_date": "2026-01-01",
  "end_date": "2026-02-09",
  "malls": ["스마트스토어", "쿠팡"],
  "page": 1,
  "per_page": 20
}
```

### POST `/api/predict-mall`

쇼핑몰별 예측

### POST `/api/predict-product`

상품별 예측

### POST `/api/export`

Excel 다운로드
